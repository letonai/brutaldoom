This is an universal file, either drag and drop, or put in your skins folder. Make sure you don`t load with other versions of this same addon.
It's made for Zdoom/GZdoom/Zandronum, but can run in other sourceports if they support .ogg files.
TNT Music can only be played ingame if loaded with Brutal Doom v21 Gold and with the Map Enhancement System turned on.
